Hassle-free deploying to GitHub Pages with Jekyll

One command, nothing more
bin/deploy "My deploy message"
Hassle-free deploying

Deploy from any GitHub branch to Github Pages with one simple command. JavaScript and CSS assets are minified and uglified using Jekyll Assets. Next to that CSS is autoprefixed for vendor browser support.

GitHub Pages hosting

Host your website for free from any of your GitHub repositories. Changes made to your GitHub Pages branch are live instantly. Perfect for blogging, project websites or documentation pages.

Easy Setup

Made for your quick prototyping needs. Kickster has an easy and unobtrusive setup which gets you started in no time. Kickster uses Jekyll so ships with endless configuration possibilities.

Jekyll plugin support

Perfect for Jekyll users that require Jekyll plugins (as GitHub Pages does not support all plugins). With Kickster you can push the output from Jekyll with any plugin (static files) to your GitHub Pages branch with one command.
