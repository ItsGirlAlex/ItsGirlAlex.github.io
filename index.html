Hassle-free deploying to GitHub Pages with Jekyll

One command, nothing more
bin/deploy "My deploy message"
Hassle-free deploying

Deploy from any GitHub branch with one simple command to GitHub Pages. JavaScript and CSS assets are minified and uglified by using Jekyll Assets. Next to that CSS is autoprefixed for multiple vendor support.

GitHub Pages hosting

Host directly for free from any of your GitHub repositories. Changes made to your GitHub Pages branch are live instantly. Perfect for blogging, project websites or documentation pages. It's possible to use custom URL's.

Easy Setup

Made for your quick prototyping needs. Easy and unobstrusive setup instructions which gets you started in no time. View the docs for the installation instructions. Uses Jekyll so ships with endless configuration possibilities.
